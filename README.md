# 藍新金流 API

本專案包含藍新金流（NewebPay）的 API 串接範例與相關文件。

## 目錄結構

- `api-example/` - API 串接範例程式
- `api-logistic/` - 物流 API 範例
- `api-platform/` - 平台管理 API 範例
- `stop_operation/` - 停運相關 API

## 使用說明

各目錄下都有對應的範例程式與說明文件，請參考各目錄內的 `readme.md` 或 `API串接範例檔使用.docx`。

## 環境需求

- PHP 7.0 以上
- Docker (可選，使用 docker-compose.yml 快速啟動)

## 快速開始

使用 Docker Compose 啟動：

```bash
docker-compose up -d
```

本專案為藍新金流 API 串接範例，僅供參考使用。
